나는 북마크 관리 페이지를 만들고자 한다.
- 브라우저 북마크에서 특정 북마크 폴더를 불러오는 기능
- 카테고리 CRUD
- 북마크 CRUD : 북마크 이름과 설명을 수정할 수 있음
- 개별 북마크는 [파비콘][북마크이름][설명]
- 개별 북마크는 드래그앤드롭으로 이동 가능하도록 (다른 카테고리로도 이동 가능하도록)
- 설정메뉴에 초기 화면을 3컬럼으로 할지, 2컬럼, 1컬럼으로 할지 설정가능
- 기본적으로 반응형으로 설계
- 깔끔하고 모던한 스타일
- lucids icon, shadcn, tailwind css 활용

===========


1.
북마크 DB를 동일 폴더에 있는 json 파일로 저장하고, 해당 json파일과 함께 호스팅에 올려서 하면,
supabase없이도 간단하게 개인이 사용할 수 있는 방식이 되지 않을까?

2. 
카테고리 순서를 변경할 수 있는 기능을 추가하고 싶다. 

3.
우측 상단의 설정 버튼의 폰트 스타일을 바로 옆에 잇는 [북마크 가져오기]와 일치하도록 변경하라.
